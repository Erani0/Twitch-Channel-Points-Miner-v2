B — Verlauf als Tabs

Jeder Run bekommt einen Tab oben. Klick auf Tab → wechselt zu diesem Run. Eingabe + Tabs kompakt zusammen.

Empfehlung: B (Tabs) — alles in einem Block, lädt schnell, klarer Wechsel zwischen Runs.
